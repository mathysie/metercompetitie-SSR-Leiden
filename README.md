# Metercompetitie SSR-Leiden

Dit programma leest de tussenstand voor de metercompetitie van SSR-Leiden uit en berekent aan de hand daarvan de kansen voor elk dispuut om een fust te winnen.

## Gebruik
Om de website van SSR-Leiden uit te lezen, gebruik je het commando `python3 tussenstand.py 1`.

Om zelf input te geven, moet je een bestand 'input.txt' in dezelfde map hebben staan als 'tussenstand.py'. Scheid daar de naam van het dispuut en het aantal meters met een komma. Bijvoorbeeld
```
L.U.C.I.F.E.R.,31
T.A.E.N.I.A.,23
M.O.C.C.A.,12
```
Gebruik dan het commando `python3 tussenstand.py`.
